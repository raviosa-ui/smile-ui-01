How this emoji affects tone and emotion

In music contexts, it is elegant and cultured.

In reaction contexts, it is highly sarcastic and dismissive.

Professional vs casual usage

In casual settings, it is a hilarious tool for sarcasm.

In professional settings, it is safe to use when discussing hobbies or attending a gala/concert.

When to use

Use this to discuss classical music, to add a dramatic flair to a sad story, or to sarcastically mock a friend who is whining about a minor inconvenience.

When not to use

Avoid using it when someone is sharing a genuine tragedy, as the 'smallest violin' trope is extremely dismissive.

Common mistakes and misuse

Using it to react to a friend's actual bad news will make you look incredibly unsympathetic.
